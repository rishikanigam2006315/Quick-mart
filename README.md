# QuickMart 🛒

QuickMart is a modern grocery delivery Android app built using Kotlin and Jetpack Compose.

## ✨ Features

- Modern Jetpack Compose UI
- MVVM Architecture
- Room Database
- Add to Cart
- Checkout Flow
- Orders Screen
- Categories
- Profile & Settings
- Dark Mode Support
- Material3 Design

## 🛠 Tech Stack

- Kotlin
- Jetpack Compose
- MVVM
- Room Database
- Navigation Component
- Material3
